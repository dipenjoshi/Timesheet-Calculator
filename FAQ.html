Frequently Asked Questions

What is this tool?

This tool is designed to help you track and calculate employee work hours, summarize them and print reports

Or track your own hours, print them, summarize them etc.

How to use?

1. Start by adding employees using the + button. Type full name and hourly rate.

2. Select an employee from the dropdown to view and edit their timesheet.

3. Add work entries by clicking on a date, type in start time & end time. Or use punch in/out buttons. It will insert current time in above field.

4. Click and drag on any dates to select a range of dates, to view summary

5. Use the Print button to print the timesheet report.

Where is my data saved?

Your data is saved locally in your browser's storage. It is not sent to any server or cloud service.

How to export the data?

There is no way to export/import data. You can print PDFs or reports and keep them for your records.
